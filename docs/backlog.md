# Product Backlog

## Níveis de Prioridade
- **1 Prioridade Alta:** Core do Produto
- **2 Prioridade Média:** Features Esperadas
- **3 Prioridade Baixa:** Features Secundárias

## Prioridade Alta

| Prioridade | Categoria | Descrição | 
|--|--|--|
|1|Cadastro|Como usuário, quero conectar-me utilizando minha conta criada durante o cadastro para ter acesso aos serviços oferecidos pelo UniJobs. |
|1|Feed Ofertas / Solicitações|Como usuário, quero ter acesso às últimas ofertas criadas para poder consultá-las.|
|1|Feed Ofertas / Solicitações|Como usuário, quero ter acesso às últimas solicitações criadas para poder consultá-las.|
|1|Busca|Como usuário, quero poder buscar por ofertas e solicitações criadas pela comunidade, filtrando-as por categoria, para achar aquelas de meu interesse.|
|1|Declaração Interesse|Como usuário, quero declarar interesse por uma oferta para ter acesso ao contato do seu oferecedor.|
|1|Criação Oferta/Solicitação|Como usuário, quero poder criar uma solicitação para informar à comunidade sobre meu interesse em um serviço.|
|1|Criação Oferta/Solicitação|Como universitário, quero poder criar uma oferta para oferecer um serviço que estou disposto a realizar.|
|1|Histórico|Como universitário, quero ter acesso a todas minhas ofertas criadas para poder consultá-las.|

## Prioridade Média

| Prioridade | Categoria | Descrição | 
|--|--|--|
|2|Cadastro|Como usuário, quero cadastrar-me utilizando um endereço pessoal de e-mail para criar uma conta na plataforma.|
|2|Cadastro|Como universitário, quero informar o meu e-mail universitário para poder confirmar que estou atrelado à uma Universidade e poder oferecer serviços através da plataforma.|
|2|Declaração Interesse|Como universitário, quero criar uma oferta à partir de uma solicitação ou selecionar uma das minhas ofertas já criadas para indicar que tenho interesse em atendê-la.|
|2|Notificações|Como universitário, quero ver as últimas notificações a respeito das minha ofertas que receberam interesse para poder ter acesso às pessoas interessadas.|
|2|Notificações|Como usuário, quero ver as últimas notificações a respeito das minhas solicitações que receberam interesse para poder ter acesso às ofertas linkadas à ela. (Obs: sempre será necessário criar uma oferta para "mostrar" interesse por uma solicitação)|
|2|Histórico|Como usuário, quero ter acesso a todas minhas solicitações criadas para poder consultá-las.|
|2|Histórico|Como usuário, quero ter acesso a todas minhas solicitações criadas para poder modificá-las ou deletá-las.|
|2|Histórico|Como universitário, quero ter acesso a todas minhas ofertas criadas para poder modificá-las ou deletá-las.|
|2|Histórico|Como universitário, quero ter acesso a todas minhas ofertas criadas para poder obter o  contato das pessoas interessadas.|
|2|Histórico|Como usuário, quero ter acesso a um histórico das ofertas pelas quais tive interesse para poder recuperar informações de contato do oferecedor.|
|2|Administração|Como administrador, quero confirmar o status de universitário de um usuário para garantir que um oferecedor de serviço pertença efetivamente à uma Universidade.|

## Prioridade Baixa

| Prioridade | Categoria | Descrição | 
|--|--|--|
|3|Cadastro|Como usuário, quero cadastrar-me com meu usuário Facebook para criar uma conta na plataforma.|
|3|Cadastro|Como usuário, quero cadastrar-me com meu usuário Twitter para para criar uma conta na plataforma.|
|3|Cadastro|Como usuário, quero ter acesso a meu perfil para poder consultar informações de cadastro e poder alterar minha foto e senha.|
|3|Feed Ofertas / Solicitações|Como usuário, quero filtrar as últimas ofertas criadas para mostrar aquelas de meu interesse.|
|3|Feed Ofertas / Solicitações|Como usuário, quero filtrar as últimas solicitações criadas para mostrar aquelas de meu interesse.|
|3|Busca|Como usuário, quero poder adicionar filtros de busca referentes à preço para refinar minha busca.|
|3|Busca|Como usuário, quero poder adicionar filtros de busca referentes à localização para refinar minha busca.|
|3|Notificações|Como universitário, quero receber notificações push quando alguém se interessar por minhas ofertas para ter acesso rápido ao contato da pessoa interessada.|
|3|Notificações|Como usuário, quero receber notificações push quando alguém linkar uma oferta à uma de minhas solicitações para verificar se tenho interesse ou não.|
|3|Notificações|Como usuário, quero ter a opção de desativar as notificações push para não recebê-las caso não tenha interesse.|
|3|Histórico|Como usuário, quero ter acesso a todas minhas solicitações criadas para poder ocultar aquelas de meu interesse para que não sejam mais mostradas na plataforma durante um intervalo de tempo.|
|3|Histórico|Como usuário, quero ter acesso à minhas solicitações criadas para poder consultar todas ofertas que foram linkadas a elas.|
|3|Histórico|Como universitário, quero ter acesso a todas minhas ofertas criadas para poder ocultar aquelas de meu interesse para que não sejam mais mostradas na plataforma durante um intervalo de tempo.|
|3|Administração|Como administrador, quero confirmar a identidade dos usuários que se cadastrarem por e-mail através do envio de SMS para garantir a segurança da comunidade.|
|3|Administração|Como moderador, quero ter acesso a todos as solicitações e ofertas criados na plataforma para poder monitorá-los e garantir a segurança e bem estar da comunidade.|
